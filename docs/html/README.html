Markdown 문서

`docs/md/`는 원본/관리용 문서를 두는 곳입니다.

문서 목록

  • `PIPELINE_OVERVIEW.md`: 전체 파이프라인 개요
  • `PIPELINE_INFRA_CHECKLIST.md`: 인프라 기능 구현 현황
  • `rag/RAG_PIPELINE_SPEC.md`: RAG 입력/출력 계약
  • `MODULE_ARCHITECTURE.md`: 모듈 구조 설명
  • `experiments/EXPERIMENT_GUIDE.md`: 실험 실행 가이드
  • `experiments/COLAB_GUIDE.md`: Colab/Drive 실행 가이드
  • `GIT_WORKFLOW.md`: Git/PR 운영 가이드
  • `KICKOFF_GUIDE.md`: 킥오프 설명 원본
  • `ROLE_GUIDE.md`: 역할 가이드
  • `TEAM_WORKFLOW.md`: 팀 운영 가이드
  • `DATA_CONTRACT.md`: 데이터 계약

HTML 대응

각 Markdown 문서는 `docs/html/`에 같은 이름의 HTML 파일을 가집니다.

예를 들어 `rag/RAG_PIPELINE_SPEC.md`는 `docs/html/rag/RAG_PIPELINE_SPEC.html`에 대응합니다.
