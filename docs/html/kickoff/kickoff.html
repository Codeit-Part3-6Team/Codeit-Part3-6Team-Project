RAG 기반 RFP 문서 분석 프로젝트

긴 입찰/RFP 문서에서 근거를 찾고, 답변과 citation을 함께 남기는 파이프라인을 만듭니다.

프로젝트 목표

문서 이해

PDF, HWP, DOCX 같은 원본 문서를 읽고 검색 가능한 형태로 나눕니다.

근거 검색

질문과 관련 있는 문서 조각을 top-k로 찾습니다.

근거 기반 답변

답변만 보여주지 않고 출처 citation을 함께 남깁니다.

평가와 발표

성공/실패 사례와 metric을 남겨 발표에서 설명할 수 있게 합니다.

일반 모델 학습과 다른 점

고전 ML 프로젝트이번 RAG 프로젝트
train/valid/test 데이터셋 중심원본 문서와 평가 질문 세트 중심
epoch, loss, checkpoint 중심chunk, retrieval, citation, 실패 분석 중심
best model weight 저장best config/index/retrieval 결과 비교
예측값이 맞는지 확인어떤 근거로 답했는지까지 확인

작업 흐름

Data 준비 -> RAG config 실험 -> 검색/답변 평가 -> 데모 형태 정리 -> 시연/발표
