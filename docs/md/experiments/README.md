# Experiment Markdown

실험 실행, Colab/Drive 실행, 노트북 사용성, 실험 운영 방법을 설명하는 Markdown 문서를 둡니다.

## 문서 목록

| 문서 | 용도 |
| --- | --- |
| `EXPERIMENT_GUIDE.md` | 실험 실행과 결과 확인 방법 |
| `COLAB_GUIDE.md` | Colab/Drive 기반 실행 방법 |
| `NOTEBOOK_USAGE_CHECKLIST.md` | 로컬 노트북과 Colab 노트북이 팀원이 쓰기 쉬운지 확인하는 기준 |

## 사용 기준

- config를 바꿔 실험할 때는 `EXPERIMENT_GUIDE.md`를 먼저 확인합니다.
- Drive 공유나 HuggingFace 계열 옵션 때문에 Colab에서 실행할 때는 `COLAB_GUIDE.md`를 확인합니다.
- 노트북 설명, metric 확인, 그래프 확인 셀을 보강할 때는 `NOTEBOOK_USAGE_CHECKLIST.md`를 기준으로 점검합니다.
